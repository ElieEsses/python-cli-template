## Installation
### Prerequisites
- [`uv`](https://docs.astral.sh/uv/) — Python package/tool manager
- `make` (optional, only needed for `make install`)
  
**Quick install:**
```bash
make install
```

**Manual:**
```bash
uv sync
uv tool install -e .
```

Verify: `devi --help`
Uninstall: `uv tool uninstall devi`
